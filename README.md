# Hermes Skills — 认知方法类

个人开发的 Hermes Agent Skills，聚焦认知方法论和知识管理。

## 安装

将 skill 目录复制到 `~/.hermes/skills/` 下即可：

```bash
# 克隆仓库
git clone https://github.com/kitsiu1211-stack/hermes-skills.git

# 复制需要的 skill
cp -r hermes-skills/ljg-rank ~/.hermes/skills/
```

或直接让 Hermes Agent 读取 GitHub URL 安装。

## Skill 列表

| Skill | 说明 |
|-------|------|
| `ljg-rank` | 领域骨架分析 — 找出背后真正支撑的几根独立的力 |
| `ljg-learn` | 深度概念解剖 — 从多维度解构任何概念 |
| `ljg-plain` | 认知原子白描 — 将内容改写为聪明中学生能懂的表达 |
| `ljg-qa` | 信息提问机 — 将文章/论文核心观点抽成 Q&A 对 |
| `ljg-card` | 内容铸造 — 将内容转化为 PNG 视觉卡片 |
| `ljg-roundtable` | 结构化圆桌讨论 — 多视角真理追寻框架 |
| `knowledge-capture` | 统一知识捕获入口 — Generator → Evaluator 双阶段 |
| `ebbinghaus-review` | 艾宾浩斯复习 — 从 Obsidian 知识图谱选取应复习的概念 |
