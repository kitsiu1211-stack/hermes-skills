# Hermes Skills 合集

19 个精选 Hermes Agent Skills，覆盖认知方法、开发实践、工具效率三大类。

## 🧠 认知方法

| Skill | 说明 |
|-------|------|
| `ljg-rank` | 领域骨架分析 — 找出背后真正支撑的几根独立的力 |
| `ljg-learn` | 深度概念解剖 — 从八维解构任何概念 |
| `ljg-plain` | 认知原子白描 — 将内容改写为中学生能懂的表达 |
| `ljg-qa` | 信息提问机 — 将文章核心观点抽成 Q&A 对 |
| `ljg-card` | 内容铸造 — 将内容转化为 PNG 视觉卡片 |
| `ljg-roundtable` | 结构化圆桌讨论 — 多视角真理追寻框架 |
| `knowledge-capture` | 统一知识捕获入口 — Generator → Evaluator 双阶段 |
| `ebbinghaus-review` | 艾宾浩斯复习 — 从 Obsidian 知识图谱选取复习概念 |

## 🛠 开发方法

| Skill | 说明 |
|-------|------|
| `systematic-debugging` | 四阶段根因调试法 — 先理解 Bug 再动手 |
| `test-driven-development` | TDD 红绿重构 — 测试先行 |
| `plan` | Plan 模式 — 模糊需求拆成 bite-sized 任务清单 |
| `spike` | Spike 探路 — 一次性快速实验验证技术方案 |
| `simplify-code` | 三 Agent 并行代码清理 — 安全 + 简化 + 测试 |
| `requesting-code-review` | 提交前自审查 — 安全扫描 + 质量门禁 |

## ⚡ 工具效率

| Skill | 说明 |
|-------|------|
| `daily-session-summary` | 每日对话摘要 — 自动汇总关键决策和待办 |
| `side-hustle-planning` | 副业冷启动规划 — 平台对比、成本测算、品类分析 |
| `weekly-deep-read` | 周度深度阅读推荐 — 基于认知版图个性化推荐 |
| `multi-agent-orchestration` | 多 Agent 编排 — 按任务类型自动分拆并行执行 |
| `chinese-search-workaround` | 中文搜索变通 — 反爬时自动切换数据源 |

## 安装

```bash
# 一键安装全部
git clone https://github.com/kitsiu1211-stack/hermes-skills.git ~/.hermes/skills-extra/

# 或单独安装某个 Skill
git clone https://github.com/kitsiu1211-stack/hermes-skills.git /tmp/hs
cp -r /tmp/hs/ljg-rank ~/.hermes/skills/
```
