# 副业冷启动规划（Side Hustle Planning）

帮助用户系统地规划跨境电商或副业项目：平台对比、成本测算、三阶段冷启动模型。

## 触发条件

- 用户提出想做副业 / 电商 / 跨境电商
- 用户问「哪个平台好」「要多少钱」「多久能回本」
- 用户讨论品类选择、供应链、投流策略

## 分析框架

### 第一步：对齐基础信息

四项必问（没对齐之前不做分析）：

1. **品类方向** — 具体到什么细分？如果没想好，先收窄
2. **平台** — TikTok Shop / Shopee / Temu / Amazon / 多平台？
3. **角色** — 全包还是合伙？有没有供应链资源？
4. **弹药** — 启动预算预期？工厂/供应链资源？

### 第二步：平台对比矩阵

### 第三步：三阶段冷启动模型

**第一阶段（第 1 月）：测品 + 测素材**

- 铺 5-10 个 SKU，小额多组投流
- 目标：找到 1-2 个有转化潜力的品
- 投流日预算：$10-15

**第二阶段（第 2 月）：爆品放大**

- 聚焦跑通的品，增加达人合作
- 目标：ROAS 从 1:1 拉到 1:2+
- 投流日预算：$15-30

**第三阶段（第 3 月）：自然流量爬坡**

- 逐步降广告占比，靠内容 + 搜索 + 复购
- 目标：70% 订单来自自然流量
- 投流日预算：$20-30

## 执行注意事项

### 供应链

- **一件代发**：1688 找供应商 → 测试出单稳定性 → 再囤货
- **囤货模式**：至少需要额外的 ¥3,000-5,000 备货资金
- **宠物用品产业带**：义乌（玩具/服饰）、邢台南和（宠物食品）、山东潍坊（宠物用品）

### 内容能力

- TikTok Shop 核心在内容，不是投流
- 宠物类目优势：不需要真人出镜，猫狗自带流量
- 每天至少 3-5 条短视频（混剪+实拍）

### 合规风险

- 宠物食品/药品有额外资质要求（FDA/进出口检疫）
- 建议从宠物玩具、服饰、用品等非食品类开始
- 各平台禁售品列表不同，先查清楚

- 不涉及具体店铺运营操作（listing 优化、广告账户设置等）
